# sphere-cli

This is a cli tool to create a new sphere project

```
Usage: sphere-cli <command> [options]

A tool for managing sphere projects
create Command:
  -help
    	show help
  -mod string
    	go module name
  -name string
    	project name
    	
rename Command:
  -help
    	show help
  -new string
    	new go module name
  -old string
    	old go mod name
  -target string
    	target directory (default ".")
```
